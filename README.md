# Claude Skills Library

A personal collection of reusable Claude skill files for AI-assisted workflows.

## What are Skills?
Each folder is a self-contained skill that Claude can load and execute. Skills are triggered by context and keywords defined in each `SKILL.md` file.

## How to Use
1. Mount this repo as your skills directory in Claude
2. Claude auto-detects and triggers skills based on your requests
3. Each skill has its own `SKILL.md` with full instructions

## Structure
```
claude-skills-repo/
├── README.md
├── skill-name/
│   ├── SKILL.md        ← Main skill instructions
│   └── references/     ← Supporting files (if any)
```
